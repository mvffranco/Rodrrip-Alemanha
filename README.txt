ROADTRIP COCKPIT v8

Como usar no GitHub Pages:
1. Crie/abra o repositório.
2. Envie todos os arquivos e pastas deste ZIP para a raiz do repositório.
3. Ative Settings > Pages > Deploy from branch > main > root.
4. Abra o link gerado pelo GitHub Pages no Safari do iPhone.
5. Toque em Compartilhar > Adicionar à Tela de Início.

Para parametrizar outra viagem:
- Opção simples: abra o app, toque em Parametrizar, altere os dados e salve no iPhone.
- Opção técnica: edite o arquivo config.json antes de subir no GitHub.
