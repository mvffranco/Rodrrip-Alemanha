# Road Trip Alemanha

App interativo publicado via GitHub Pages.

## Como publicar
1. Suba todos os arquivos deste ZIP no repositório.
2. Confirme que o arquivo principal se chama `index.html`.
3. Em Settings > Pages, use:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
4. Acesse:
   `https://mvffranco.github.io/Rodrip-Alemanha/`

## Como instalar no iPhone
Abra o link no Safari > Compartilhar > Adicionar à Tela de Início.
